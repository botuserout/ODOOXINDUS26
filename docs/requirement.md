# Odoo X Indus Hackthon'26

## Goal
Recreate the provided landing page design exactly with full responsiveness and identical UI behavior.

## Core Requirements
- Pixel-close layout to reference design.
- Responsive across:
  - Desktop
  - Laptop
  - Tablet
  - Mobile
- Smooth animations and interactions.
- Cross-browser compatibility.

## Technology Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- jQuery
- Animate.css
- Slick Carousel
- Magnific Popup
- FontAwesome / IcoFont
- Canvas API (particle effects)
- Parallax + Tilt effects

## Functional Requirements
- Fixed responsive navigation bar
- Gradient hero section
- Animated headings
- CTA buttons with hover effects
- Background canvas animation
- Parallax depth effects
- Mobile menu support
- Smooth scrolling

## Non-Functional Requirements
- Page load under 2s on broadband
- Mobile-first responsiveness
- Clean folder structure
- Maintainable code
